### Sobre

Página para visualização de dados do futebol brasileiro. Foco em campeonatos brasileiros e confrontos entre os times.

### Descrição

No site você irá encontrar:

- **Tabelas detalhadas dos campeonatos brasileiros;**

- **Histórico de eliminações** - saiba os detalhes de TODOS confrontos eliminatórios (ou finais) entre os times do G12;

- **Tabela acumulada** dos pontos corridos (você pode fazer recortes por data);

- **Histórico de confrontos** entre times;

- **Maiores goleadas** - filtre por quantidade de gols, times, etc;

- **Baixe os dados de maneira simples e use em seus projetos!**
